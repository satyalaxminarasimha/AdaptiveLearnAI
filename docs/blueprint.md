# **App Name**: AdaptiveLearnAI

## Core Features:

- AI Quiz Generation: Generates quizzes based on syllabus topics and student performance using LLMs.
- Personalized Learning Paths: Provides tailored learning paths based on student quiz results and identified weak areas, enhanced by AI analysis.
- Weak Area Analysis: Analyzes student performance to identify weak areas, leveraging AI to provide targeted recommendations.
- Role-Based Dashboards: Provides role-specific dashboards (Admin, Professor, Student) with relevant information and actions.
- Real-Time Chat: Enables real-time chat between students, professors, and AI assistants using Firestore.
- Request & Approval Workflow: Allows users to submit requests (e.g., syllabus changes) and admins to approve or reject them.
- AI Chat Tutor: Allows students to ask questions and receive explanations, recommendations, and study plans, guided by the student's performance data; uses an LLM tool for deciding whether and when to inject student's previous performance data into its reasoning.

## Style Guidelines:

- Primary color: Deep purple (#673AB7) to evoke a sense of knowledge and sophistication.
- Background color: Light gray (#F5F5F5), a very desaturated purple, for a clean and modern look.
- Accent color: Blue (#2196F3) to highlight interactive elements and calls to action.
- Body and headline font: 'Inter', a sans-serif font known for its clean readability and modern aesthetic.
- Use a consistent set of icons to visually represent subjects, topics, and features within the app.
- Design the layout with a focus on providing information hierarchically to assist learning; avoid clutter to focus attention on key elements; create consistent visual spacing between elements.
- Subtle animations to indicate progress and provide feedback to users.
